### Fast-snake
Really fast snake - all operation with O(1) time.

### How to build
clone repo:
```
git clone https://github.com/Mikhaeris/fast-snake.git
```

goto directory:
```
cd fast-snake
```

build:
```
make release
```

run:
```
./build/release/main
```

#### One command:
```
git clone https://github.com/Mikhaeris/fast-snake.git
cd fast-snake

```
